# Wedding Guest Management Portal

A sleek, modern administrative dashboard frontend built with HTML5 and CSS3. This portal serves as a centralized hub for managing wedding guest lists, dynamically linking users to various Google Sheets segments (Overall List, Statistics, Bride's Guests, and Groom's Guests).


##  Key UI/UX Features

- Modern Gradient Aesthetics: Features dynamic linear gradients and interactive micro-interactions (`hover` animations with subtle translations and custom box shadows).
- Component isolation: Leverages CSS pseudo-elements (`::before`) to cleanly inject thematic emojis without cluttering the semantic HTML structure.
- Unified Branding: Uses an elegant, content-appropriate palette built around royal purples (`#6f42c1`), pastel pinks (`#ffe5ef`), and deep slate accents (`#4d4260`).

---

##  Technical Highlights

- CSS Flexbox Architecture: Utilized flexible column and row flows to guarantee accurate card centering and uniform button spacing across all devices.
- Dynamic CSS Transitions: Applied `transition: all 0.3s ease` to provide smooth visual feedback on user actions.
- Integration Readiness:Implemented secure external redirection targets (`target="_blank"`) for external data analytics tools.
